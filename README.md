# Portfolio Project – Junior Growth Marketing Specialist

## Tools Installed

- Cursor IDE – Downloaded and installed from [cursor.com](https://cursor.com/)
- Claude Code (Cursor Extension) – Installed via the Extensions panel in Cursor (Ctrl+Shift+X) and logged in
- Codex (Cursor Extension) – Installed via the Extensions panel in Cursor (Ctrl+Shift+X) and logged in

## Steps Completed

1. Downloaded and installed Cursor IDE from the official website
2. Opened Cursor and navigated to the Extensions panel using Ctrl+Shift+X
3. Searched for "Claude Code", installed the extension, and logged in
4. Searched for "Codex", installed the extension, and logged in
5. Created a public GitHub repository on github.com
6. Cloned the repository into Cursor using Git Clone via the Command Palette (Ctrl+Shift+P → Git: Clone)
7. Created this README.md file to document the process
8. Committed and pushed the file to GitHub

## Issues Encountered & How I Solved Them

- Finding the Extensions panel: I initially looked in the wrong place — I opened the Agent Marketplace inside Cursor which shows MCP plugins and skills. After some searching, I discovered that the Extensions panel is a separate section, accessible by pressing Ctrl+Shift+X or clicking the editor window sidebar. Once I found the correct panel, I was able to search for and install both Claude Code and Codex successfully.
